Shadowsocks 客户端教程

Jun 28, 2019 · kopever

Shadowsocks 服务端基于 shadowsocks-libev + v2ray 搭建的客户端使用教程

Windows

  1. 下载 shadowsocks-windows 最新版客户端；
  2. 下载 shadowsocks-v2ray-plugin 最新版客户端插件；
  3. 解压 shadowsocks-windows 客户端；
  4. 解压 shadowsocks-v2ray-plugin 插件；
  5. 将 v2ray-plugin.exe 放到解压的 shadowsocks-windows 客户端的目录；
  6. 打开 shadowsocks-windows 目录的客户端程序 Shadowsocks.exe；
  7. 填写服务器配置地址、端口、密码、加密；
  8. 插件程序栏填入插件程序的文件名 v2ray-plugin。

macOS

  1. 下载 ShadowsocksX-NG 客户端；
  2. 解压 ShadowsocksX-NG.zip；
  3. 打开 ShadowsocksX-NG 并填入服务器、端口、密码、算法；
  4. 插件选择 v2ray-plugin。

iOS

  1. 进入 AppStore 下载 Shadowrocket 最新版客户端程序 (美区)；
  2. 打开 Shadowrocket 并点击右上角加号添加服务器节点；
  3. 类型选择 Shadowsocks 并填入服务器、端口、密码、算法；
  4. 点击插件，选择 v2ray-plugin 并填入地址及端口 (同上)；
  5. 返回点击右上角"完成"。

Android

  1. 下载并安装 shadowsocks-android 最新版客户端；
  2. 下载并安装 v2ray-plugin-android 最新版客户端插件；
  3. 打开 Shadowsocks 客户端并填入服务器、端口、密码、加密方式；
  4. 插件选择 v2ray。
